# GTR
Go TraceRoute


This is a simple [MTR](http://www.bitwizard.nl/mtr/) clone in Go 


Usage
-----
```go get github.com/mag-/gtr```

```cd $GOPATH/src/github.com/mag-/gtr```

```go build```

```sudo ./gtr google.com```
